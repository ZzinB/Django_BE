# 생활 폐기물 수집 노인’을 위한 솔루션 ‘할매 여깃소’
- 생활 폐기물사진을 올리면 종류를 분류하고, 위치를 알려주는 웹 프로젝트

  ### 역할
  Backend
  - AI 모델 연결
  - 파일 업로드
  - 업로드 된 이미지의 메타정보를 데이터베이스에 저장
  - AI 모델 결과 Label 데이터베이스에 저장
 
  ### 배운점 및 아쉬운 점
  - 
  - 디테일이 아쉬웠음
  - Frontend와의 협업 (API 설명서의 중요성)
  - AI 파이프라인 연결이 아쉬웠음
